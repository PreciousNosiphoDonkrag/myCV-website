Projects

Year Project: Monitoring the Water Hyacinth on the Hartbeespoort Dam

Description: Collaborated with the Belgium Campus and Rhodes University to develop an end-to-end solution for monitoring the invasive water Hyacinth species on the Hartbeespoort Dam. The project involved gathering requirements, designing a machine learning algorithm, and creating an application for researchers and fishermen to predict, locate, and track the spread of the species.

Automating Loan Eligibility Process

Description: Developed a machine learning model to automate the loan eligibility process for a financial institution. This project emphasized understanding the business process, data exploration, and model optimization to classify loan applicants effectively. Key Contributions: Conducted Exploratory Data Analysis (EDA) to understand and preprocess the data. Experimented with various algorithms and conducted hyperparameter tuning to optimize model performance. Implemented data transformation and feature engineering to improve model accuracy. Deployed the model for real-time classification. Technologies Used: Python, Scikit-Learn, Pandas, NumPy.

Handwritten Digit Recognition

Description: Developed an algorithm to classify handwritten digits (0-9) from the MNIST dataset. This project was a deep dive into image processing and machine learning techniques, emphasizing data preprocessing and model training. Key Contributions: Preprocessed the MNIST dataset, including flattening and normalizing the images. Designed and implemented an ml model to classify digits. Evaluated model performance and fine-tuned parameters to achieve optimal results. Documented the entire process and findings, highlighting challenges and solutions. Technologies Used: Python, TensorFlow, Keras, Pandas, NumPy.
